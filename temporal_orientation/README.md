# Prospection Lexicon: Temporal Orientation

This archive contains a single lexicon for Temporal Orientation.

To learn how to use this lexicon, please refer to the [paper](http://wwbp.org/publications.html#p37). 

Note that the intercept (marked as "_intercept" in the lexica csvs) is to be added post weighted lexicon extraction.
